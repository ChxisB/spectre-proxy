Modify files using exact string replacement.

Options:
- `filePath`: Path to the file to edit (required)
- `oldString`: The text to replace (required) — include LINE#HASH| prefix from read output for strict verification
- `newString`: The replacement text (required)
- `replaceAll`: Replace all occurrences (optional)

HASHLINE SUPPORT: When using line references from Read output (LINE#HASH|), 
the tool verifies the content hasn't changed since you read it. If the file
was modified, you'll receive corrected line references — retry with those.
